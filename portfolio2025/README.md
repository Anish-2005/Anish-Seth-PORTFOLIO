## üì¶ Installation

1. Clone the repository:
```bash
git clone https://github.com/Anish-2005/Portfolio2025.git
cd portfolio2025
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## üé® Customization

### Update Personal Information

Edit the following files to customize with your information:

- `components/Hero.tsx` - Update name, title, and social links
- `components/About.tsx` - Modify the about description
- `components/Skills.tsx` - Add/remove skills and adjust proficiency levels
- `components/Projects.tsx` - Add your projects with descriptions and links
- `components/Contact.tsx` - Update contact information and email

### Color Scheme

The color scheme uses Tailwind CSS utilities. Main colors:
- Primary: Blue (`blue-400`, `blue-500`)
- Secondary: Purple (`purple-400`, `purple-500`)
- Background: Slate (`slate-900`, `slate-800`)

Update `tailwind.config.ts` to customize the theme.
